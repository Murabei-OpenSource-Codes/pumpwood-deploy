 57                # Azure:

 60                # GCP:
 61                - credential_file: Set a path to a credetial file of a service
 62                    user with access to the bucket that will be used at the
 63                    deployment.
 64                # AWS:
 65                - aws_access_key_id: Access key of the service user with
 66                    access to the s3 used in deployment.
